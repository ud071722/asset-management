# 주택 구매 가능성 계산기 (Asset Management Dashboard)

현재 가용 자산과 대출 가능액을 기반으로 주택 구매 가능성을 시뮬레이션하는 웹 도구입니다.

## 주요 기능
- **취득 비용 자동 계산**: 매매가에 따른 취득세 및 중개 수수료 자동 계산.
- **가용 자산 분석**: 보유 주택 매매 예상가, 현금, 기타 수입 등을 통합 관리.
- **대출 및 상환 시뮬레이션**: 필요 대출액 산출 및 예상 월 납부금(원리금균등) 계산.
- **참고 시세 관리**: 관심 있는 아파트 단지의 시세를 목록으로 관리 및 비교.
- **데이터 저장**: 브라우저 로컬 저장소(`localStorage`)를 활용한 자동 저장 기능.

## 기술 스택

- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
