# 🏠 주택 구매 가능성 계산기 프로젝트 가이드

이 프로젝트는 현재의 자산과 대출 가능액을 분석하여 주택 구매 가능성을 실시간으로 계산하고, 클라우드에 데이터를 저장하는 웹 애플리케이션입니다.

---

## 🚀 주요 기능
1. **실시간 자산 계산**: 목표 주택 가격, 세금, 부대비용 및 현재 자산을 입력하면 즉시 부족 자금과 대출 필요액을 계산합니다.
2. **대출 시뮬레이션**: 금리와 기간에 따른 원리금 균등 상환액 및 이자 비용을 계산합니다.
3. **참고 시세 관리**: 관심 있는 아파트들의 시세를 리스트로 관리하며, 구매 가능 여부를 토글로 확인할 수 있습니다.
4. **Firebase 클라우드 동기화**: '기록 저장하기' 버튼을 누르면 어떤 기기에서 접속하든 동일한 데이터를 불러올 수 있습니다.
5. **Netlify 자동 배포**: GitHub에 코드를 올리면 즉시 인터넷 주소에 반영됩니다.

---

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (ES6 Modules)
- **Database**: Firebase Firestore (v9 Modular SDK)
- **Deployment**: Netlify + GitHub

---

## 📂 파일 구조 설명
- **`index.html`**: 웹사이트의 뼈대입니다. 화면 레이아웃과 버튼, 입력창들이 정의되어 있습니다.
- **`style.css`**: 디자인의 핵심입니다. 세련된 프리미엄 디자인(Glassmorphism)과 가로 배치 레이아웃이 설정되어 있습니다.
- **`app.js`**: 모든 로직의 중심입니다. 계산 공식, Firebase 연동, 추가/삭제 기능이 담겨 있습니다.
- **`README.md`**: 프로젝트의 기본적인 소개 파일입니다.

---

## 🔄 수정 및 업데이트 워크플로우
만약 기능을 추가하거나 디자인을 바꾸고 싶다면 다음 단계를 따르세요:

1. **로컬 수정**: 컴퓨터에서 `app.js`나 `style.css`를 수정합니다.
2. **GitHub 업로드**: 수정된 파일을 GitHub 저장소에 업로드(Commit)합니다.
3. **Netlify 확인**: 약 1분 뒤에 [배포된 주소]에 접속하여 반영된 내용을 확인합니다.

---

## ⚠️ 주의사항 및 팁
- **로컬 실행**: 파일을 그냥 더블 클릭해서 열면 Firebase와 JavaScript 모듈이 작동하지 않습니다. 반드시 **배포된 주소**에서 확인해야 합니다.
- **데이터 저장**: 입력을 마친 후에는 반드시 우측 상단의 **[기록 저장하기]** 버튼을 눌러야 클라우드에 반영됩니다.
- **삭제 기능**: 참고 시세에서 항목을 클릭(선택)하여 파란색 테두리가 생긴 것을 확인한 뒤, 상단의 **[- 삭제]** 버튼을 누르세요.

---

## 🌟 향후 발전 방향
- **로그인 기능 추가**: 사용자마다 본인만의 데이터를 따로 관리할 수 있게 업그레이드할 수 있습니다.
- **추이 그래프**: 자산 변화나 시세 변화를 그래프로 시각화하는 기능을 추가할 수 있습니다.
- **공인중개사 연결**: 특정 매물에 대해 바로 문의할 수 있는 링크 기능을 넣을 수 있습니다.
